# AI Module - Document Summarization & Guidance

Python-based module for:
- Document text extraction
- NLP-based summarization
- Key point extraction
- Suggestion generation

## Setup

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Module Structure

- `models/` - Trained/pre-trained models
- `scripts/` - Processing and training scripts
- `data/` - Sample data for testing

## Key Features

- PDF/Document text extraction
- Extractive & abstractive summarization
- Named entity recognition
- Key point extraction
- Action suggestion generation

## API Integration

Communicates with backend API via HTTP endpoints.
